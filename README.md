🛒 Unisole – E-commerce Website

🚀 A full-stack e-commerce web application for buying shoes online.
Built using modern web technologies with authentication, cart system, and responsive UI.

---

🌐 Live Demo

🔗 https://unisole.onrender.com

---

📌 Features

- 🔐 User Authentication (Login & Register)
- 🛍️ Add to Cart functionality
- 📦 Product Management
- ⚡ REST API with Node.js & Express
- 🗄️ MongoDB Database Integration
- 🎨 Responsive UI Design

---

🛠️ Tech Stack

Frontend:

- HTML
- CSS
- JavaScript

Backend:

- Node.js
- Express.js

Database:

- MongoDB

UNISOLE/

⚙️ Installation

1. Clone the repository

git clone https://github.com/aakashkoli330-cloud/UNISOLE.git

2. Install dependencies

npm install

3. Run the server

npm start
